# 人脸识别考勤系统

## 一、系统核心功能需求

### 1. 学生信息管理

- **人脸注册**：支持录入学生基本信息（学号、姓名、班级），并通过摄像头采集至少3-5张不同角度的人脸图像作为注册样本，提取特征值存入数据库。
- **库管理**：实现对班级学生信息的增、删、改、查功能。

### 2. 考勤打卡与识别

- **实时检测**：调用摄像头实时捕获画面，检测画面中的人脸位置。
- **身份识别**：将捕获到的人脸与数据库中已注册的人脸进行特征比对，确定学生身份。
- **考勤记录**：识别成功后，系统自动记录学生的姓名、学号、识别时间（签到时间）以及考勤状态（如“正常”、“迟到”）。

### 3. 活体检测（可选）

- 能区分真实人脸与照片、视频或手机屏幕的翻拍。可采用简单的指令动作（如眨眼、张嘴）或基于纹理分析的方法实现。

### 4. 考勤数据管理（教师端）

- **课程管理**：教师可以创建课程，并关联参与该课程的学生名单。
- **考勤报表**：按课程、按日期、按学生维度生成考勤统计表，如出勤率、缺勤名单、迟到记录等。
- **数据导出**：支持将考勤记录导出为Excel或CSV格式，便于存档或打印。
- **异常处理**：支持教师手动修改考勤结果（如补签、请假审批）。

### 5. 学生端/查询功能（可选）

- **考勤查询**：学生可通过学号和密码登录系统，查看自己的历史考勤记录。
